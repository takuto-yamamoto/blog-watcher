# Architecture Decision Records (ADRs)

This directory contains Architecture Decision Records documenting key technical decisions made during the development of blog-watcher.

## What is an ADR?

An Architecture Decision Record (ADR) captures an important architectural decision made along with its context and consequences. ADRs help maintain a historical record of why certain choices were made.
